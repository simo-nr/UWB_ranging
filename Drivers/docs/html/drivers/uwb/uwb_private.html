void dwt_writetodevice(uint32_t regFileID, uint16_t index, uint16_t length, uint8_t *buffer)

This function is used to read from the UWB IC (e.g., DW3000, QM33xxx) device registers.

Parameters:

  • regFileID – [in] : ID of register file or buffer being accessed

  • index – [in] : Byte index into register file or buffer being accessed

  • length – [in] : Number of bytes being read

  • buffer – [out] : Pointer to buffer in which to return the read data.

Returns:

None

void dwt_readfromdevice(uint32_t regFileID, uint16_t index, uint16_t length, uint8_t *buffer)

This function is used to write to the UWB IC (e.g., DW3000, QM33xxx) device registers.

Parameters:

  • regFileID – [in] : ID of register file or buffer being accessed

  • index – [in] : Byte index into register file or buffer being accessed

  • length – [in] : Number of bytes to write

  • buffer – [in] : Pointer to buffer which contains the data to write.

Returns:

None
